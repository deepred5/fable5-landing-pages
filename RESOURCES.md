# Web Motion & Interaction Design Resources

## Knowledge

- [Course: Animations on the Web — Emil Kowalski](https://animations.dev/)
  Linear 设计工程师(前 Vercel)的交互式动效课程,业内公认的"动效品味"第一课。第一模块 "Making it Feel Right" 讲 easing、速度感知、spring 的理论。Use for: 动效理论、easing 直觉、React/Motion 组件动效。
- [Docs: GSAP 官方文档 + Learning Center](https://gsap.com/resources/)
  框架无关动效库的事实标准,文档配 20 万+ 帖子的官方论坛。Use for: timeline 编排、ScrollTrigger、ease 可视化(内置 Ease Visualizer)。
- [Tutorials: Codrops (tympanus.net)](https://tympanus.net/codrops/)
  创意开发社区的核心教程站,大量 Awwwards 级案例的逐步拆解(scroll、WebGL、shader、页面过渡)。Use for: 实战复刻、了解"获奖动效是怎么写出来的"。
- [Course: Three.js Journey — Bruno Simon](https://threejs-journey.com/)
  93 小时的 WebGL/three.js 体系课程,从场景搭建到 shader、物理、Blender,公认的 WebGL 入门唯一推荐。Use for: 路线第四阶段(WebGL/3D),前三阶段不要碰。
- [Docs: Motion (原 Framer Motion)](https://motion.dev/)
  React 生态动效标准库官方文档,含 variants、layout、scroll、exit 动画。Use for: React 项目里的产品级动效。
- [Docs: MDN — CSS scroll-driven animations](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_scroll-driven_animations)
  原生滚动驱动动画(animation-timeline: scroll()/view())的权威参考。Use for: 路线第一阶段,零依赖实现滚动动效。

## Wisdom (Communities)

- [GSAP 官方论坛](https://gsap.com/community/)
  高活跃、有 GSAP 团队成员答疑。Use for: 动效实现卡壳、性能问题、代码评审。
- [Awwwards](https://www.awwwards.com/)
  创意网站评奖平台。Use for: 建立眼界基准(每周看 SOTD)、目标校准;后期可投稿自己的作品。
- [Three.js Journey Discord](https://threejs-journey.com/)
  课程附带的会员制 Discord,WebGL 学习者聚集地。Use for: 进入 WebGL 阶段后的提问与作品反馈。
- [Codrops 案例作者访谈](https://tympanus.net/codrops/category/interviews/)
  creative developer 的成长路径访谈(如 Adrián Gubrica 从学生到 SOTD 得主)。Use for: 职业路径参考。

## Gaps
