三条铁律

  • 入场 → ease-out 族(先快后慢):用户在等,起步要快,落点要稳。
  • 退场 → ease-in 族(先慢后快)或短到没人注意:离开的元素不值得戏份。
  • linear 只用于机械运动:跑马灯、扫描线、loading 旋转、进度。

性格曲线用途

机械匀速linear跑马灯、CRT 扫描带、雷达、无限循环位移

经验法则

  • cubic-bezier(a, b, c, d):a/b 管起步,c/d 管刹车;b 或 d 超出 [0,1] = 回弹。
  • 同一页面统一一族曲线 = 风格一致性的最大来源(对照上表)。
  • UI 微交互 150–300ms;区块入场 500–800ms;氛围/水墨 1s+。先定性格,再调时长。
  • 调试工具:Chrome DevTools 的 Animations 面板可点击曲线图标实时拖控制点。
